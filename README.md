# Custom Gradle Task

## Overview

## Requirements

* Java, prerably OpenJDK 13
* [Gradle](https://gradle.org/) 6.0.1 or newer

## Setup

Clone the Gradle Project locally. Execute `gradle tasks --all`. This will compile
the Custom Task Library and include it in your classpath.

## Usage

```bash
gradle tasks --all
grade greeting
```
